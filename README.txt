PROYECTO: PANIFICADORA LA MERCED

Tecnologías utilizadas:
- Python
- Tkinter
- MySQL

Descripción:
Aplicación de escritorio que se conecta a una base de datos MySQL
para mostrar información de pedidos y clientes mediante una
interfaz gráfica.

Instrucciones:
1. Instalar MySQL Server
2. Ejecutar el script crear_bd.sql
3. Activar el entorno virtual
4. Ejecutar: python app_panificadora.py



docker-compose down

docker-compose down -v

docker-compose up -d --build
docker-compose up -d 

docker ps

docker logs panificadora_db
